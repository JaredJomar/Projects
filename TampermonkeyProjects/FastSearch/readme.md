## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Click [here](#) to install the script (link coming soon)
3. Press Insert key to start using

## Keyboard Controls

| Key | Function |
|-----|----------|
| Insert | Open search interface |
| ESC | Close interface |
